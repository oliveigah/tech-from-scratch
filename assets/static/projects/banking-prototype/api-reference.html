    Banking Prototype v0.1.0 API Reference

    Modules

    Account

    Pure functional module that manages Account data structures

    Account.Cache

    DynamicSupervisor that manages all Account.Server processes currently running on the system

    Account.Exchange

    Module responsible for the currency conversion logic and the maintance of the exchange rates table

    Account.Operation

    Pure functional module, used to define new operations that happen over an Account data structure

    Account.Server

    GenServer that keeps track of an Account data structure as it's state

    Database

    Supervisor that manages Database.Worker processes, responsible for data persistence functionality and workers pooling

    Database.Worker

    GenServer that handle data persistance requests

    Metrics.Collector

    Module responsible for gathering system's metrics and persist it on the database

    Metrics.Scheduler

    GenServer responsible for start the Metrics.Collector task every 5 minutes
